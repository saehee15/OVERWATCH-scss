### 반복문
#### @for $i from 1 through 32 {
#### &:nth-child(#{$i}) .image{
#### background-image: url("./images/hero#{$i}.png");
#### }
---
### list와 map 은 반복문 @each
#### $list: orange, pink, yellow;
#### $map: (o: orange,p: pinky: yellow)
#### @each $c in $list {
#### .box{color: $c;}
#### }
#### @each $K, $v in $map{
#### .box-#{$k} {color: $v;}
#### }
